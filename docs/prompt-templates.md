## 5. 通用建筑打印约束词
建议追加到任何建筑类 prompt 后面：

- monolithic geometry
- printable silhouette
- low-poly
- reduced detail
- thickened structure
- stable base
- no micro details
- no fragile cantilever
- prototype scale
